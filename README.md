# Counter App (React)

---

This project is a **counter application** built with **React JS**, created to practice state management using the `useState` hook.

---

## Features

-   Increment the counter by **+1**.
-   Decrement the counter by **-1**, preventing negative values.
-   Reset the counter to the initial value (**0**).
-   Simple and clean UI with custom styling.

---

## Concepts Covered

-   State management with `useState`.
-   Updating state based on the previous value.
-   Event handling (`onClick`).
-   Functional components in React.

---

## Technologies Used

-   React JS
-   JavaScript (ES6+)
-   CSS / Tailwind CSS

---

## Project Purpose

This project is part of my React learning process, focused on understanding how state works, how components re-render, and how users interact with the UI.
